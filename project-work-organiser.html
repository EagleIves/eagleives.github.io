Dynamic Work Organiser

An advanced Excel-based task management solution with VBA automation, dynamic named ranges, and intelligent workflow tracking

2024
Productivity Tool
Excel, VBA

What Is It?

A simple yet powerful to-do list tracker and dashboard that provides a comprehensive view of pending tasks, responsible parties, and upcoming deadlines. This isn't just another spreadsheet – it's an intelligent workflow management system built entirely in Excel.

This document utilises several advanced Excel tools and features, namely: Visual Basic macros, dynamic named ranges, conditional formatting, and sheet navigation buttons. Update your list of completed tasks with the click of a button and watch your progress unfold automatically.

Features & Tools

1

Interface Dashboard

The Interface serves as the command center, summarising all tasks according to their deadline and current status. Features include:

  • • Page navigation buttons to full To Do List, Completed Tasks, and standardised list entries
  • • Automatic updates reflecting changes in work statuses
  • • Visual summary of tasks by priority and deadline
  • • Quick access to all key sections of the organiser

Main dashboard interface showing task summaries and navigation buttons

2

To Do List

The heart of the system, detailing all ongoing workstreams and tasks to be completed.

Smart Deadline Highlighting:

  • • Amber highlighting for tasks due this week
  • • Red highlighting for overdue tasks
  • • Automatic archiving when tasks marked as complete

Tasks marked as complete are automatically copied to the "Completed" sheet for archiving, triggered by changing the status from "In progress" to "Completed".

To Do List with color-coded deadline highlighting (amber and red)

3

Completed Tasks Archive

An automated archiving system that maintains a historical record of completed work:

  • • Automatic copying of completed tasks from the To Do List
  • • Sort button to arrange tasks by deadline in ascending order
  • • Provides performance tracking and accountability records
  • • Useful for progress reports and retrospective analysis

Completed tasks archive with sort functionality

4

Dynamic Lists Management

The backend configuration page that powers the entire system's flexibility:

  • • Hosts named ranges and data validation lists
  • • Any additions automatically added to saved named ranges
  • • Ensures data consistency across all sheets
  • • Easy customization for different team structures and workflows

Lists page managing standardised entries for projects and statuses

Key Benefits

  • Zero Manual Tracking: Automated status updates and archiving eliminate the need for manual task management.
  • Visual Priority Management: Color-coded deadlines make it immediately clear which tasks need urgent attention.
  • Historical Records: Automatic archiving creates a complete audit trail of completed work for performance reviews.
  • Team Collaboration: Clear responsibility assignments and status visibility enable better team coordination.
  • Easy Customization: Dynamic named ranges and standardised lists make it simple to adapt to any workflow.
  • No Additional Software: Built entirely in Excel – no subscriptions, installations, or learning curves required.
